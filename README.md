# TXG2TPL
An (un)packer for Naruto GNT4's TXG files to a more editable format.

# Usage
+ Unpacking: TXG2TPL -u <TXG File>.txg
+ Creating: TXG2TPL -p <Folder Path> <Output Name>.txg
